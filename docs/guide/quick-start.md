# 快速开始

本指南帮助您在 5 分钟内完成 NetProxy 的基本配置。

## 第一步：添加节点

通过手动添加 Xray 出站配置添加代理节点：

1. 进入 **节点** 页面
2. 点击右上角 **添加** 按钮
3. 选择 **添加出站**
4. 输入文件名和 Xray outbound JSON 内容
5. 点击 **保存**

## 第二步：选择节点

1. 在节点列表中找到要使用的节点
2. 点击节点卡片进行切换
3. 当前选中的节点会显示高亮状态

## 第三步：启动代理

1. 进入 **状态** 页面（首页）
2. 点击大的 **启动/停止** 按钮
3. 服务启动完成
